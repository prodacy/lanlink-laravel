Vaga de Desenvolvimento

Faça o clone do repositório no local junto aos seus projetos

Abra o terminal da sua máquina

Navegue até a pasta do clone

Com o Composer instalado

Execute:

$ composer install

$ php artisan migrate --seed

$ php artisan serve
